# 📜 卷軸機率模擬器

楓之谷風格的卷軸使用模擬器，可追蹤各種卷軸的成功、失敗與摧毀次數，並記錄連續與累計數據。

---

## 快速開始

直接用瀏覽器開啟 `index.html` 即可使用，不需要任何安裝或伺服器。

---

## 使用方法

### 1. 選擇卷軸
點擊上方的卷軸圖示來選擇要使用的卷軸，目前選中的卷軸會以金色邊框高亮顯示。

### 2. 使用卷軸
點擊「**使用**」按鈕進行一次模擬，結果會以彈出提示顯示：
- ✅ 成功
- ❌ 失敗
- ❌ 道具摧毀

### 3. 查看紀錄
下方會顯示各卷軸的統計卡片，包含：
- 成功 / 失敗 / 摧毀 次數
- 實際成功率
- 連續成功 / 連續失敗
- 累計成功 / 累計失敗

### 4. 重置
每張統計卡片下方有「🔄 重置」按鈕，可清除該卷軸的所有紀錄。

## 介面功能

### 顯示紀錄卡片
主面板下方有各卷軸的勾選框，可以控制哪些統計卡片要顯示。
- 點擊卷軸圖示時會自動勾選並顯示對應卡片
- 目前使用中的卷軸卡片無法被隱藏

### 固定卷軸欄
往下捲動頁面時，頂部會自動出現一排固定的卷軸選擇欄與「使用」按鈕，方便不用捲回頂部就能操作。

### 進階設定（右側面板）
點擊畫面右側的「◀」按鈕展開進階設定：

| 設定 | 說明 |
|------|------|
| 點擊冷卻時間 | 每次使用卷軸之間的最短間隔（0–600 ms）|
| 音效音量 | 調整成功、失敗、按鈕音效的音量（0–100%）|

## 備註

- 所有紀錄會自動儲存在瀏覽器的 `localStorage`，重新整理後不會遺失
- 機率使用 `Math.random()` 產生，每次獨立隨機，無任何保底機制
